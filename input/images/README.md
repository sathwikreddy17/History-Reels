# Images Directory

This is where you organize images for your stories.

## Folder Structure
Create one folder per story with the story name:
```
images/
├── Epic of Gilgamesh/
│   ├── gilgamesh1.jpg
│   ├── gilgamesh2.png
│   └── enkidu.webp
├── hercules-twelve-labors/
│   ├── hercules.jpg
│   └── nemean_lion.jpg
└── Perseus and Medusa/
    ├── perseus.jpg
    └── medusa.png
```

## Naming Guidelines
- Folder name should match your story file name
- System handles flexible naming (spaces, dashes, underscores)
- Examples:
  - Story: `Epic of Gilgamesh.txt` → Folder: `Epic of Gilgamesh/`
  - Story: `hercules_labors.txt` → Folder: `hercules-labors/` ✅

## Image Requirements
- **Formats**: `.jpg`, `.jpeg`, `.png`, `.webp`
- **Quantity**: 2-8 images per story (optimal: 4-6)
- **Quality**: High resolution recommended (1080p+)
- **Content**: Mythology-themed, story-relevant visuals

## Tips for Best Results
- Use high-quality, mythology-themed artwork
- Include character portraits and scene imagery
- Avoid modern or irrelevant stock photos
- Consider the story flow when selecting images

## Ken Burns Effects
The system automatically applies cinematic pan and zoom effects to your static images, creating dynamic video content.

For detailed setup instructions, see [QUICK_START.md](../../QUICK_START.md) in the main directory.
