# Stories Directory

This is where you place your story files for processing.

## Supported Formats
- `.txt` files (recommended)
- `.docx` files (requires python-docx package)

## File Naming
You can name your files however you like:
- `Epic of Gilgamesh.txt`
- `hercules_twelve_labors.txt`
- `Perseus-and-Medusa.txt`

## Content Guidelines
- Any length story (system adapts to 60-second format)
- Rich, descriptive content works best
- Mythology, spiritual stories, legends perform well
- Visual storytelling elements enhance results

## Example Stories
Check the main content/scripts/ folder for examples of generated scripts to see the output format.

## Usage
1. Add your story file here
2. Create matching image folder in `../images/`
3. Run `python process_stories.py`
4. Get your video in `renders/` folder

For detailed instructions, see [QUICK_START.md](../../QUICK_START.md) in the main directory.
