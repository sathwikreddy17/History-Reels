# Outro Video Requirements

## Outro Clip
- Add a 2-3 second outro video to this folder
- Resolution: 1080x1920 (9:16 vertical)
- Format: .mp4
- Content: Call-to-action like "Follow for more ancient stories"

## Typical CTAs:
- "Follow for more mythology"
- "Like and subscribe"
- "What myth should we explore next?"
- Social media handles

## Sample filename:
- outro.mp4

The system will automatically use the first .mp4 file found in this folder.
