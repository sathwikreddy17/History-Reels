# Curated Images for Mythology Videos

## 🎯 **Perfect Solution for Mythology Content**

This directory contains curated images for each mythology topic. This approach is **much better** than generic stock videos because:

- ✅ **Specific content** - Gods, mythological scenes, ancient art
- ✅ **You control quality** - Hand-picked, relevant images  
- ✅ **Perfect storytelling** - Each image relates directly to the script
- ✅ **Authentic visuals** - Real mythological artwork vs. generic footage

## 🖼️ **How It Works**

1. **Add images** to the topic folder (3-8 images per topic)
2. **System auto-detects** images and creates video
3. **Each image shows** for 2-3 seconds with Ken Burns effect
4. **Smooth transitions** with fade in/out
5. **Perfect sync** with AI voiceover

## 📋 **Image Guidelines**

### **Best Image Types:**
- Ancient art/paintings of gods and goddesses
- Historical artwork depicting mythological scenes
- Temple architecture and sacred sites
- Symbolic objects (weapons, artifacts, etc.)
- Classical paintings from museums

### **Technical Requirements:**
- **Format:** JPG, PNG, WEBP
- **Resolution:** Minimum 1080px wide
- **Aspect:** Any ratio (will be auto-cropped to 9:16)
- **Quality:** High resolution for better zoom effects

### **Content Guidelines:**
- **Authentic:** Real mythological artwork
- **Relevant:** Matches the specific story
- **High-quality:** Clear, detailed images
- **Diverse:** Different scenes/characters per story

## 🚀 **Adding Your Images**

### Step 1: Create Topic Folder
```bash
mkdir "assets/images/Your_Topic_Name"
```

### Step 2: Add Images
Place 3-8 relevant images in the folder:
- `your_topic_1.jpg`
- `your_topic_2.jpg` 
- `your_topic_3.jpg`
- etc.

### Step 3: Run Generator
```bash
python main.py
```

The system will automatically:
- Detect your curated images
- Create video with Ken Burns effects
- Sync with AI-generated script and voiceover
- Generate professional Instagram Reel

## 🎬 **Video Generation Priority**

1. **🖼️ Curated Images** (BEST - this folder)
2. **🎥 Local Videos** (assets/videos/)
3. **📱 Pexels API** (fallback for generic content)

## 💡 **Pro Tips**

- **3-5 images** work best for 60-second videos
- **Mix wide and close-up shots** for variety
- **Include character images** and scene backgrounds
- **Add symbolic elements** (weapons, artifacts, etc.)
- **Use museum-quality artwork** when possible
