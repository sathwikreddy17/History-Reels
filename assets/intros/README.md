# Intro Video Requirements

## Intro Clip
- Add a 2-3 second branded intro video to this folder
- Resolution: 1080x1920 (9:16 vertical)
- Format: .mp4
- Content: Your brand logo, channel name, or simple title card

## Creation Tools:
- Canva (templates available)
- After Effects
- DaVinci Resolve
- Online tools like Lumen5

## Sample filename:
- intro.mp4

The system will automatically use the first .mp4 file found in this folder.
