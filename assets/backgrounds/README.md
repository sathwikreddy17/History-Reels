# Asset Requirements

## Background Images
- Add 10-15 vertical images (1080x1920) to this folder
- Themes: Ancient temples, landscapes, mythological scenes
- Formats: .jpg, .png
- The system will randomly rotate through these with Ken Burns effects

## Recommended Sources:
- Pexels (free)
- Unsplash (free) 
- Stock photo sites
- AI-generated images (MidJourney, DALL-E)

## Sample filenames:
- ancient_temple_01.jpg
- mystical_forest_02.jpg
- golden_sunset_03.jpg
- etc.
